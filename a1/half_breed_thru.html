Brace Thru

(formerly Half Breed Thru)

From facing couples, each composed of a man and a lady: All Right Pull By. Those dancers (if any) in a normal couple Courtesy Turn to end facing the other couple. Those in a half sashayed couple U-Turn Back, turning toward each other. Ends in normal facing couples.

Note: In 2012 the name of this call was changed from Half Breed Thru to Brace Thru. The new name was chosen because the final action is the call Brace Yourself (which is not on the Advanced program). The CALLERLAB Advanced Committee requests that all callers adopt and use only the new name by April, 2013.

Timing: 6, SS 8
